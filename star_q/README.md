# star_q

**星问——让每个“小问号”都能起飞** 公益服务平台

​	

## 技术栈
- vue-CLI

- vue-router  路由

- vuetify  UI框架

- Vee-validate  验证框架
  ​	

## 项目启动
`/star_q` 文件夹下

1. 安装依赖

    ```
    $ npm install
    ```

    ​	

2. 项目运行

    ```
    $ npm run serve
    ```

    ​	

3. 访问地址

    Local: http://localhost:8080/

    Network: http://192.168.2.102:8080/
